\### Real-World Connection

Used by system administrators and security analysts to quickly gather system information during maintenance or incident response.



| \*\*Bash scripting\*\* | Used to automate penetration testing, log analysis, and system enumeration |

| \*\*System info collection\*\* | Attackers use similar scripts to gather system details before privilege escalation |

| \*\*Defenders\*\* | Use these scripts to quickly assess system state during incident response |



System administrators:

• Inventory machines

• Troubleshoot systems

• Verify configurations



Attackers:

• Enumerate the host after gaining access

• Identify privilege escalation paths

• Discover valuable information



Defenders:

• Collect triage information

• Verify compromised systems

• Speed up incident response
