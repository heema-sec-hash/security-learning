## Commands Learned Today

| Command | Use |
|---------|-----|
| `find` | Search for files by size, user, group, etc. |
| `grep` | Search for text inside files |
| `sort` | Sort lines |
| `uniq -u` | Show unique lines (only once) |
| `strings` | Extract human-readable text from binary files |
| `\|` | Pipe (send output from one command to another) |
| `2>/dev/null` | Hide error messages |
